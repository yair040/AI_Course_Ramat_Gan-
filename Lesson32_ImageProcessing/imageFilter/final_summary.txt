## Command-Line Interface

### Updated Commands

```bash
# All filters (HPF, LPF, BPF)
python main.py --input photo.jpg --filter all

# Individual filters
python main.py --input photo.jpg --filter hpf --hpf-cutoff 35
python main.py --input photo.jpg --filter lpf --lpf-cutoff 25
python main.py --input photo.jpg --filter bpf --low-cutoff 30 --high-cutoff 70
```

### New Parameters
- `--hpf-cutoff FLOAT` - HPF cutoff (default: 30.0)
- `--lpf-cutoff FLOAT` - LPF cutoff (default: 30.0) **NEW**
- `--low-cutoff FLOAT` - BPF lower (default: 20.0)
- `--high-cutoff FLOAT` - BPF upper (default: 80.0)

### Removed Parameters
- ~~`--cutoff FLOAT`~~ - Split into `--hpf-cutoff` and `--lpf-cutoff`
