# ============================================================================
# Installation Notes
# ============================================================================
#
# 1. Create virtual environment:
#    python3 -m venv venv
#    source venv/bin/activate
#
# 2. Upgrade pip:
#    pip install --upgrade pip
#
# 3. Install requirements:
#    pip install -r requirements.txt
#
# 4. Verify installation:
#    python -c "import anthropic; import pdfplumber; import chromadb; print('OK')"
#
# 5. If torch installation fails, try:
#    pip install torch --index-url https://download.pytorch.org/whl/cpu
#
# 6. If ChromaDB installation fails, try:
#    pip install chromadb --no-cache-dir
#
# ============================================================================
# Expected Installation Size
# ============================================================================
# - Total download: ~2-3 GB
# - Installed size: ~4-5 GB
# - PyTorch is the largest package (~1-2 GB)
# - Transformers models will download on first use (~100-500 MB)
#
# ============================================================================
# Minimum System Requirements
